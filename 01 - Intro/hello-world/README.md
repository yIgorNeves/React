Conceitos basicos do React:
1 - Como criar o projeto: npx create-react-app my-app
    my-app sera substituido pelo nome desejado. Não utilizar espaços.
2 - Iniciar o projeto apos a criação: npm start
    Ira iniciar o servidor com a tela padrão do React
3 - Editar o arquivo App.js para edição da tela principal.
4 - Editar o arquivo index.html para edição do titulo da pagina e outros.
